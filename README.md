# E-Commerce Sales Analysis

## Project Overview
This project analyzes the Olist Brazilian E-Commerce dataset to identify sales trends, top-performing product categories, customer satisfaction levels, and regional sales performance.

The analysis includes data cleaning, exploratory data analysis, visualizations, dashboard creation, and business recommendations.

---

## Objectives

- Analyze sales performance over time
- Identify top-performing product categories
- Compare sales across states/regions
- Evaluate customer review scores
- Create a management dashboard
- Generate business insights and recommendations

---

## Dataset

Dataset Used:
Olist Brazilian E-Commerce Dataset (Kaggle)

- olist_orders_dataset.csv

## Analysis Questions

### 1. Which product category generated the highest revenue?
**Answer:** Beauty & Health (beleza_saude)

### 2. Which month recorded the highest sales?
**Answer:** November 2017

### 3. Which state generated the highest revenue?
**Answer:** São Paulo (SP)

### 4. What is the Average Order Value trend?
AOV remained relatively stable with seasonal fluctuations.

### 5. What is the review score distribution?
Most customers provided 5-star ratings.

---

## Key Business Insights

1. Beauty & Health products generated the highest revenue.
2. November 2017 was the strongest sales month.
3. São Paulo contributed the largest share of revenue.
4. Customer satisfaction is generally high.
5. Low review scores indicate opportunities for service improvements.
